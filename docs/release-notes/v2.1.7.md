## 中文

### Anthropic Messages 协议适配 + 第三方模型 reasoning 修复

新增 `apiFormat=anthropic_messages`,将 Codex CLI Responses 请求转换到 Anthropic `/v1/messages`,并把 Anthropic Messages SSE 还原为 Responses SSE。

修复 Kimi/DeepSeek 等第三方模型在 Codex CLI 0.130 主界面不显示 thinking 的问题。

主要改动:

- Anthropic Messages 请求/响应完整映射:text、thinking、tool_use、tool_result、input_image、input_file、server_tool_use、web_search_tool_result 等 content block。
- 工具映射:function tool、namespace tool、custom tool、Anthropic hosted tools(bash、computer_20250124、str_replace_editor、web_search_20250124 等)。
- 强制 tool-call 时自动移除 `thinking` 和 `output_config.effort` 以避免 Anthropic 429/520,同时保留 forced `tool_choice`。
- 原生 `session cache` 支持多轮对话继续,无需本地状态累积。
- invalid thinking signature 自动重试,保障 tool-round 连续性。
- 通用第三方模型 catalog 模板默认 `supports_reasoning_summaries: true`;Apply Config 自动写入 `model_supports_reasoning_summaries = true`;Restore 按快照恢复。
- Antigravity 卡片临时隐藏(现有配置不受影响)。

## English

### Anthropic Messages protocol + third-party model reasoning fix

New `apiFormat=anthropic_messages` converts Codex CLI Responses API requests to Anthropic `/v1/messages` and maps Anthropic Messages SSE back to Responses SSE.

Fixes Kimi/DeepSeek and other third-party models not displaying thinking in Codex CLI 0.130 main UI.

Changes:

- Full Anthropic Messages request/response mapping: text, thinking, tool_use, tool_result, input_image, input_file, server_tool_use, web_search_tool_result content blocks.
- Tool mapping: function tools, namespace tools, custom tools, Anthropic hosted tools (bash, computer_20250124, str_replace_editor, web_search_20250124).
- Removes `thinking` and `output_config.effort` on forced tool-call to avoid Anthropic 429/520, while keeping forced `tool_choice`.
- Native `session cache` for multi-turn conversation continuation without local state accumulation.
- Automatic retry on invalid thinking signature to preserve tool-round continuity.
- Generic third-party model catalog template defaults to `supports_reasoning_summaries: true`; Apply Config writes `model_supports_reasoning_summaries = true`; Restore respects snapshot.
- Antigravity card temporarily hidden (existing configs unaffected).

### Verification

- `cargo fmt --check`
- `cargo test -p codex-app-transfer-codex-integration`
- `cargo test -p codex-app-transfer-adapters anthropic_messages`
- End-to-end: Anyrouter forced tool-call + continuation probe passed
